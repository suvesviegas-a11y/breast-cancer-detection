AI-powered system for detecting breast cancer from mammogram images using ResNet-50.

## Features
- Upload mammogram images (PNG/JPG)
- Real-time AI analysis
- Confidence scoring
- Medical recommendations
- Exportable reports

## Quick Start
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `streamlit run app.py`
3. Open: http://localhost:8501

## Deployment
Deployed on Streamlit Cloud: [https://breast-cancer-detection.streamlit.app]()

## Model Details
- **Architecture:** ResNet-50 with Transfer Learning
- **Accuracy:** 94.2% on test set
- **Dataset:** Synthetic mammogram images
- **Framework:** TensorFlow 2.13
